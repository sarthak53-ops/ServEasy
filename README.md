
# ServEasy

ServEasy is a **Smart Food Demand Prediction System** designed to help restaurants and food outlets manage their operations efficiently.
The system combines **billing, inventory management, and supplier notifications** with future integration of **AI-based demand prediction**.

The goal of ServEasy is to reduce food wastage, optimize stock levels, and automate supply management using data-driven insights.

---

## Features

* Billing system for order processing
* Inventory tracking for ingredients and food items
* Supplier notification when stock levels are low
* Storage of sales data for future demand prediction
* Simple and easy-to-use interface

---

## Technologies Used

* **Node.js** – Backend server
* **JavaScript** – Core application logic
* **HTML** – User interface
* **JSON** – Data storage for inventory and sales data

---

## Project Structure

server.js – Main backend server
billing.html – Billing interface
inventoryManager.js – Handles inventory operations
inventory.json – Inventory database
sales_data.json – Sales records used for prediction
package.json – Project dependencies

---

## How to Run the Project

### 1. Clone the repository

git clone https://github.com/sarthak53-ops/ServEasy.git

### 2. Install dependencies

npm install

### 3. Start the server

npm start

### 4. Open the application

http://localhost:3000
